nachtanken
==========

Die Turtle durchsucht alle 16 internen Slots
nach brauchbarem Treibstoff und setzt diesen
ein. 

Damit der Treibstoff beim Einpacken/Abbauen
der Turtle erhalten bleibt, muss die Turtle
einen Namen haben. Das wird automatisch
ueberprueft.

Der Name kann mit "label set [name]" oder mit
"computername" eingestellt werden.
